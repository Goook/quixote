
## 为什么要学习数据结构与算法
1. 不只是用轮子，能够造轮子
2. 提升大脑的思维能力

#### 基础20个常用的数据结构与算法。
* 10个数据结构： 数组、链表、队列、栈、散列表、二叉树、堆、跳表、图、trie树。
* 10个算法： 递归、排序、二分查找、搜索、哈希算法、动态规划、回溯算法、分治算法、贪心算法、字符串匹配算法

#### 学习算法必须问的几个问题
1. 他的来历
2. 他解决的问题
3. 自身的特点
4. 实际的应用场景



### 一些技巧
1. 边学边练，适度刷题
2. 多问、多思考、多互动
3. 知识需要沉淀，不要一下子掌握所有
