###6. ZigZag Conversion

```
The string "PAYPALISHIRING" is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

P   A   H   N
A P L S I I G
Y   I   R

And then read line by line: "PAHNAPLSIIGYIR"

Write the code that will take a string and make this conversion given a number of rows:

string convert(string s, int numRows);
Example 1:

Input: s = "PAYPALISHIRING", numRows = 3
Output: "PAHNAPLSIIGYIR"
Example 2:

Input: s = "PAYPALISHIRING", numRows = 4
Output: "PINALSIGYAHRPI"
Explanation:

P     I    N
A   L S  I G
Y A   H R
P     I

```

###Answer
根据规律可知，对于Row固定时，每个字符在第几行的位置是固定的，且是相对有序的。\
因此可以根据所在row进行分类，最后进行拼接

```python
class Solution(object):
    def convert(self, s, numRows):
        """
        :type s: str
        :type numRows: int
        :rtype: str
        """
        row = 0
        direct = 1
        dic = {i: [] for i in range(numRows)}
        for c in s:
            dic[row].append(c)
            row, direct, numRows = self.judge(row, direct, numRows)
        return "".join(map(lambda x: "".join(dic[x]), dic))



    @staticmethod
    def judge(row, direct, numRows):
        if row == numRows:
            direct = -1
        elif row == 0:
            direct = 1
        row += direct
        return row, direct, numRows

```


###Fastest
```python
class Solution(object):
    def convert(self, s, numRows):
        """
        :type s: str
        :type numRows: int
        :rtype: str
        """
        if numRows == 1 or numRows >= len(s):
            return s

        L = [''] * numRows
        index, step = 0, 1

        for x in s:
            L[index] += x
            if index == 0:
                step = 1
            elif index == numRows -1:
                step = -1
            index += step

        return ''.join(L)
```
